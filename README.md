# Ignite Tech Solutions - Report Preview Board

A lightweight, high-visibility report status board built with React, Tailwind CSS, and Vite.

## Deployment to Vercel

This project is optimized for Vercel. You can deploy it in one click:

1.  **Push to GitHub**: Push this code to a GitHub repository.
2.  **Import to Vercel**: Go to [vercel.com](https://vercel.com) and import the repository.
3.  **Configuration**:
    *   **Framework Preset**: Vite
    *   **Build Command**: `npm run build`
    *   **Output Directory**: `dist`
4.  **Environment Variables**: (Optional) If you later integrate a backend, add your keys in the Vercel dashboard.

## Features

- **Live Board**: Infinite scrolling status board for floor monitors.
- **Admin Dashboard**: Secure management of reports (`/admin`).
- **Vibrant Theme**: High-contrast, minimal design with glowing accents.
- **Responsive**: Works on all screen sizes.
